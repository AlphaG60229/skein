Skein

A single-file, local-first viewer for a folder of Markdown notes. It reads notes/*.md, parses YAML frontmatter, and draws the [[wikilinks]] between notes as an interactive graph. Nothing ever leaves your machine.
Your browser will ask for permission to read the folder — that prompt is the security boundary, so a page can never scan your disk silently. Write access is only requested later, the first time you save.
Works in Chromium browsers (Chrome, Edge). Firefox and Safari do not support the File System Access API yet.
